    Save and Load
    RHEOS.savedata — Method.
    savedata(self::RheologyData; filedir::String = "", ext = "_RheologyData.jld")

    Save RheologyData object using JLD format. Save file directory must be specified. If data was loaded from disk using fileload or the full RheologyData constructor then filedir argument can be set to empty string "" which will try to use the original file dir concatenated with the optional ext string argument - e.g. "/originalpathto/file.csv_RheologyData.jld".
